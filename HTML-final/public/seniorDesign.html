Senior Design

This project was a senior design project for Group 31

Group 31: An interactive, three-dimensional visual reconstruction of a whole E. coli cell

Team Members: Richard (Yuxuan) Xie, Rohan Grover, Winnie Shi
Mentor: Nathan Mih
Advisor: Bernhard Ø. Palsson

for full list of people who were involved in this project please visit About > About Us

+Expand all

This project aims to build a structural visual reconstruction of E. coli and provide a pipeline to allow expansion to other organisms. The model will focus on the structure and proteomics of E. coli and will be displayed on a clean, easy-to-use website for end users to interact with. Interact and explore our model on this website and enjoy.

A visual model is a valuable tool that provides  insights that a graph or a table cannot. The global impact of this model resides in the pipeline created for generating similar models of its kind for other organisms and conditions at the cellular scale. For research, this project acts as a stepping stone for creating more accurate and dynamic models for the "inside-out" approach towards cellular modelling. For education, this model would have the most immediate impact in providing a more realistic view of the complexity and crowding in the cell. It would also be a valuable tool to view structural information and strengthen the connection between structural and systems biology. With this project, we hope to help people gain a better idea of the cellular environment and inspire future improvements for modelling and research.

There were no costs with this project. All the tools used were open source or free to use for UCSD students.

Parts, Software, Resources Details
Individual laptops or computers Used to develop the model and analysis
Protein Databases Protein structures and protein complex information will be provided by PDB, EcoCyc, and GEM-PRO databases. Unknown structures could be potentially predicted using SWISS-MODEL tool
Protein interaction data Data will be collected from EcoCyc and Biosystems Laboratory
cellPACK Software to pack and build the 3D models developed by researchers at Scripps
IMOD Software for tomogram analysis developed by University of Colorado in Boulder
Tomograms and Membrane Images Collected from various different papers and database
NGL Viewer Used to display our model on the website
Cinema4D and Blender Cinema4D was used with cellPACK for building the model. Blender was used for building the membranes
SQL database Used to build database to store structural, stoichiometric, localization, PPI, and metabolic data
Python Used to automate processes and collect data
Heroku Used for website backend and frontend integration

The main ethical concern would be to give credit to the information and data sources used, the people who designed the software tools (NGL viewer, cellPACK, etc.), and the people who have helped us put together this project. The software used were all open source.

Due to the nature of this project, there were no health, safety, and environmental concerns in regards to chemicals, biologics, electronics, mechanics. There was the concern in regards to making sure that the data available on our website is secure for the users who are interested to download. Because the website does not require the user to submit any private or sensitive information, we should be safe in those regards.

  • Richard: The senior design project provides great opportunities to learn new skills and to think independently and critically as a researcher. Since the project is focused on the computational side, biophysical, computational, and systems biology approaches are applied, which are not tangible in the designed track of bioengineering specialization. Programming and modeling are the skills sets that I think I will benefit the most in the future graduate study. I have used and integrated various open source softwares along with the postgreSQL database to build a platform that enables the combination of data generation, analysis, and display, and these techniques are universal and can be applied in other discipline as well.

  • Rohan: This project exposed me to a variety of computational biology techniques that I was never aware of, as well as whole sides of the discipline I had never really touched. Working with ssbio gave me deeper insight into the interactions between systems and structural biology and helped me better understand ways to construct and interpret biological networks. Additionally, I developed a better understanding as to why we need visualizations, especially after working with other fragmented, inconsistent knowledgebases. I got to hone my Python and teamwork skills and develop completely new ones, such as full-stack web development and design. I’m extremely grateful for the opportunities this project has afforded me and hope I can continue to apply this newfound knowledge.

  • Winnie: Through this project, I became more comfortable with the basics in Python, html, and css as well as learning how to use different softwares such as Blender and IMOD. It has also deepened my understanding of systems biology and the current knowledge on E. coli. Throughout this project, there was continuous learning, but there was also a sense of unknowns. E. coli is one of the most studied organisms and yet we still understand so little and there is so much more we can explore and research in the future. The senior design aspect taught me how to write better, present the work, and organize the pieces that go into the project. This project has been very rewarding and I am very grateful to have been part of this team.
