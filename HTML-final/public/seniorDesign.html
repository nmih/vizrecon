Senior Design

This project was a senior design project for Group 31

Group 31: An interactive, three-dimensional visual reconstruction of a whole E. coli cell

Team Members: Richard (Yuxuan) Xie, Rohan Grover, Winnie Shi
Mentor: Nathan Mih
Advisor: Berhard Ø. Palsson

for full list of people who were involved in this project please visit About > About Us

Overview of Project Solution

This project aims to build a structural visual reconstruction of E. coli and provide a pipeline to allow expansion to other organisms. The model will focus on the structure and proteomics of E. coli and will be displayed on a clean, easy-to-use website for end users to interact with. Interact and explore our model on this website and enjoy.

Tools and Costs

There were no costs with this project. All the tools used were open source or free to use for UCSD students.

Parts, Software, Resources Details
Individual laptops or computers Used to develop the model and analysis
Protein Databases Protein structures and protein complex information will be provided by PDB, EcoCyc, and GEM-PRO databases. Unknown structures could be potentially predicted using SWISS-MODEL tool
Protein interaction data Data will be collected from EcoCyc and Biosystems Laboratory
cellPACK Software to pack and build the 3D models developed by researchers at Scripps
IMOD Software for tomogram analysis developed by University of Colorado in Boulder
Tomograms and Membrane Images Collected from various different papers and database
NGL Viewer Used to display our model on the website
Cinema4D and Blender Cinema4D was used with cellPACK for building the model. Blender was used for building the membranes
SQL database Used to build database to store structural, stoichiometric, localization, PPI, and metabolic data
Python Used to automate processes and collect data
Heroku Used for website backend and frontend integration

Environmental, Ethical, Health and Safety Issues

The main ethical concern would be to give credit to the information and data sources used, the people who designed the software tools (NGL viewer, cellPACK, etc.), and the people who have helped us put together this project. The software used were all open source.

Due to the nature of this project, there were no health, safety, and environmental concerns in regards to chemicals, biologics, electronics, mechanics. There was the concern in regards to making sure that the data available on our website is secure for the users who are interested to download. Because the website does not require the user to submit any private or sensitive information, we should be safe in those regards.
